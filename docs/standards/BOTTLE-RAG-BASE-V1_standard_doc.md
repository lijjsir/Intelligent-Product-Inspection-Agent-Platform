# 瓶罐容器检测标准 (BOTTLE-RAG-BASE-V1)

## 1. 外观检测

### 1.1 瓶身裂缝 (bottle.exterior.crack)
- **判定标准**: 瓶身任何位置出现贯穿性或可见裂缝，无论长度，直接判定为 **不合格**。
- **检测方法**: 目视检测，强光透射辅助。
- **严重度**: 致命 (critical)
- **处置**: fail

### 1.2 瓶身划痕 (bottle.exterior.scratch)
- **判定标准**: 深度超过0.1mm或长度超过5mm的划痕需 **人工复核**；轻微表面擦伤可放行。
- **检测方法**: 目视检测 + 手感触摸。
- **严重度**: 轻微 (minor)
- **处置**: manual_review

## 2. 瓶口检测

### 2.1 瓶口变形 (bottle.opening.deformation)
- **判定标准**: 瓶口椭圆度超出标准公差（±0.3mm），影响密封性能，判定为 **不合格**。
- **检测方法**: 专用量规测量。
- **严重度**: 严重 (major)
- **处置**: fail

### 2.2 瓶口螺纹损伤 (bottle.opening.thread_damage)
- **判定标准**: 螺纹出现缺口、毛刺或塌陷，无法正常旋盖，判定为 **不合格**。
- **检测方法**: 目视 + 旋盖测试。
- **严重度**: 严重 (major)
- **处置**: fail

## 3. 标签检测

### 3.1 标签缺失 (bottle.label.missing)
- **判定标准**: 瓶身应贴标签区域无标签或标签完全脱落，判定为 **不合格**。
- **检测方法**: 目视检测。
- **严重度**: 严重 (major)
- **处置**: fail

### 3.2 标签印刷模糊 (bottle.label.print_blur)
- **判定标准**: 标签文字、条形码或二维码模糊到无法扫描/识读，需 **人工复核**。
- **检测方法**: 扫码枪验证 + 目视。
- **严重度**: 轻微 (minor)
- **处置**: manual_review

## 4. 瓶底检测

### 4.1 瓶底沉淀/异物 (bottle.bottom.sediment)
- **判定标准**: 瓶底有可视沉淀物或异物（适用于透明/半透明瓶体），判定为 **不合格**。
- **检测方法**: 翻转目视检测。
- **严重度**: 严重 (major)
- **处置**: fail

### 4.2 瓶底变形 (bottle.bottom.deformation)
- **判定标准**: 瓶底不平导致放置不稳定（倾斜>3°），判定为 **不合格**。
- **检测方法**: 水平台面测试。
- **严重度**: 严重 (major)
- **处置**: fail
